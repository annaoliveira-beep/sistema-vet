# Projeto: PetCare - Sistema de Clínica Veterinária

## 📌 Apresentação para Portfólio

### Descrição Executiva
PetCare é uma aplicação web completa para gerenciar pacientes em clínicas veterinárias. Desenvolvida como projeto de portfólio, demonstra habilidades em desenvolvimento full-stack com Python, Flask, e SQLite3.

### Objetivos do Projeto
- Demonstrar conhecimento em desenvolvimento web
- Imprimir design UX/UI moderno
- Implementar sistema seguro de autenticação
- Aplicar boas práticas de código
- Criar documentação profissional

## 💼 Caso de Uso

### Usuário Final
Uma clínica veterinária que precisa gerenciar seus pacientes de forma eficiente, mantendo dados seguros e organizados.

### Fluxo Típico
1. Gerente cria conta
2. Registra todos os pacientes
3. Visualiza lista organizada
4. Faz atualizações conforme necessário
5. Deleta registros obsoletos
6. Faz logout seguro
